How to get your Anthropic API key

Hyrd uses the Anthropic API to generate your resume and cover letter. You need your own key to run it. This takes about two minutes and costs nothing to set up.

1
Create an Anthropic account
Go to console.anthropic.com and sign up with your email address. If you already use Claude.ai, you can sign in with the same account.

2
Add billing information
Go to Settings, then Billing, and add a credit or debit card. Anthropic charges per API call, not a monthly subscription. For personal use with Hyrd, you will spend well under a dollar for dozens of runs. Set a monthly spend limit if you want a hard ceiling.

Setting a spend limit of $5 to $10 per month is more than enough for regular personal use and prevents any surprise charges.
3
Generate your API key
Click API Keys in the left sidebar, then Create Key. Give it a recognisable name like "hyrd" so you know what it is later. Copy the key the moment it appears. Anthropic only shows it once.

Copy the key now. This is the only time Anthropic will show it to you. If you lose it, you will need to create a new one.
4
Paste it into Hyrd
Click the API Key button in the top right corner of Hyrd, paste your key into the field, and hit Save. Your key stays in the browser for that session only. When you close the tab, it is gone. You will need to paste it again next time you open the app.

What it will cost you

Each time you generate a resume and cover letter, Hyrd sends your job description and resume to the Anthropic API and gets back the tailored content. You are billed for the text processed, not per session.

Keeping your key safe

Treat your API key like a password. Do not paste it into websites you do not control, do not share it in messages or screenshots, and do not commit it to a public GitHub repository. In Hyrd it travels only from your browser directly to Anthropic. No other server sees it.
